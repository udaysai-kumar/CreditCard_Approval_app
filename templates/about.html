About the project

AI Credit Card Approval Prediction System

Meridian is an end-to-end banking decision-support system that predicts whether a credit-card application should be approved or rejected, using supervised machine learning trained on applicant demographic, financial, and credit-history data.

Data pipeline

Duplicate removal, missing-value handling, feature engineering (age and employment tenure from raw date fields), and categorical encoding — mirroring production-grade preprocessing.

Model selection

Four classifiers are trained and scored on accuracy, precision, recall, F1, and ROC-AUC. The highest-F1 model is automatically promoted to production.

Secure by design

Passwords are hashed with Werkzeug's PBKDF2 implementation; sessions are server-side and cookie-scoped; every prediction is tied to an authenticated user.

Full audit trail

Every applicant record and prediction outcome is persisted to SQLite and available for search, filtering, sorting, and CSV export.

Disclaimer

This is an academic / portfolio demonstration project built for an internship submission. It is trained on synthetically generated data that mirrors the structure of the public "Credit Card Approval Prediction" dataset, and is not connected to any real banking system or bureau. It should not be used for actual lending decisions.
